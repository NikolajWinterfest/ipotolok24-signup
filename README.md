### Цель проекта:

Реализовать форму регистрации юридического лица с валидацией и интеграцией с Dadata.ru API.

### Задачи проекта:

#### Создать страницу регистрации юридического лица с полями:

– Имя контактного лица<br />
– Email<br />
– Пароль<br />
– Телефон<br />
– ИНН<br />

#### Интеграция с Dadata.ru:

– Автозаполнение данных компании при вводе ИНН<br />
– Показ индикатора загрузки во время запроса<br />
– Обработка ошибок (неверный ИНН, проблемы с сетью)<br />

#### Валидация:

– Проверка всех полей на стороне клиента<br />
– Понятные сообщения об ошибках<br />
– еактивация кнопки отправки при наличии ошибок<br />

#### Отправка формы:

– Логирование данных в консоли при успешной валидации<br />
– Показ уведомления об успешной отправке<br />
– Обработка ошибок при отправке<br />

#### UX/UI:

– Дебаунс запросов к Dadata<br />
– Адаптивная верстка<br />
– Чистый и читаемый интерфейс<br />

## Запуск проекта

### Склонировать проект на ваш компьютер с Github с помощью команды:

```
gh repo clone NikolajWinterfest/ipotolok24-signup
```

### Установить зависимости с помощью команды

```
npm install
```

### Запустить проект с помощью команды

```
npm run dev
```

### Собрать проект с помощью команды

```
npm run build
```
